== Installation ==

1. Uploader le fichier allopass.zip et decompressez le dans le dossier /wp-content/plugins/wp_allopass/
2. Activez the plugin dans la rubrique extension de WordPress
3. Paramétrez votre compte Allopass (Réglages)
